### Installation

1. clone this repo
   ```sh
   git clone https://github.com/acele-happy/ATPL_Portifolio_Backend
   ```
2. Install packages
   ```sh
   npm install / npm i
   ```
3. Run in DEV mode
   ```sh
   npm start
   ```

4. Run tests
    ```sh
    npm test
    ```
